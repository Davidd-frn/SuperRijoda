Description de votre projet

Par Fournier David, Dozinel Jonathan, Murtezani Rinor, le 2 octobre

Super Rijoda est un jeu de plateforme inspiré du classique Super Mario. Le joueur incarne Rijoda, un héros devant parcourir différents niveaux remplis d’obstacles, d’ennemis et de bonus afin de sauver son monde.

Le projet a pour but de recréer un univers rétro en 2D avec des mécaniques simples : sauter, courir, ramasser des pièces et éviter les pièges. De plus, nous voulons ajouter quelques nouveautés propres à notre jeu pour le rendre unique.

Ce projet est réalisé dans le cadre du cours de Web Development HES-SO Valais.

Le code source est disponible sur notre GitHub.
